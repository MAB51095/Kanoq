# Kanoq
Simple accounts application for Screen Printers

AIM : To convert the /Document/Accounts.xlsm to simple application to allow more flexibility in
  - Client CRUD
  - Tailor CRUD
  - Models CRUD
  - Payment CRUD
  - Income CRUD
  - Reporting
  

Application layer

Frontend - 
  - ReactJs
Backend - 
  - API using .Net 
  - SQL Server as  DB
